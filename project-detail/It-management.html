Brief Description:This IT Management System, built on Laravel, is a comprehensive web application designed to streamline various IT and business management tasks. It includes two portals: an admin portal and an employee portal, each offering tailored features based on user roles and permissions.

Dasboard Features

Admin Dashboard: A centralized overview of key metrics and activities across clients, employees, and projects.

Login page for admin and employees

Employee

Features for adding employees, viewing detailed profiles, and managing departmental designations.

Client

Capabilities for adding clients, viewing client profiles, and associating projects with specific clients.

Project

A dedicated section to oversee ongoing projects, track progress, and manage client interactions.

Department

Organizational structure with department listings and employee designations.

Expenses

A module for recording and managing company expenses.

Service Management

Service categories and service projects displayed on the website via API integration.

Web Support

A system for storing and managing customer/client requests, such as custom website development or pre-made website purchases.

Invoice Generation

Allows for the generation of invoices for random or specific clients, with options to email and download in PDF format.

Role Management

A module for managing user roles and permissions across different sections of the application.

Employee Dasboard Features

Employee Dashboard: The employee portal provides a dashboard and role-based access to projects, tasks, and other relevant sections, ensuring employees see only the data they need.
